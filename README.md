## Стек:

- JavaScript
- NODE
- Express
- MongoDB
##

### Установка

`npm install` - установить зависимости проекта

`npm run dev` - запуск сервера с hot-reload на [localhost:3000](localhost:3000)

`npm run start` - запуск сервера на [localhost:3000](localhost:3000)
